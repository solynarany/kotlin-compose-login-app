
# Мобильное приложение авторизации (Kotlin + Jetpack Compose)

## 📌 Описание проекта
Данное мобильное приложение разработано в рамках учебного курса по методическим материалам и представляет собой систему регистрации и авторизации пользователей. В ходе работы я самостоятельно спроектировала интерфейс экранов, логику валидации данных и архитектуру приложения.

## 🛠 Технологический стек
* **Язык программирования:** Kotlin
* **UI-фреймворк:** Jetpack Compose (современный декларативный подход к UI)
* **Архитектура:** Компоненты Jetpack, управление состояниями экранов (State management).
* **Стилизация:** Кастомная цветовая палитра (BackgroundPink, PrimaryPink), шрифты и кастомные элементы UI.

## 🚀 Реализованный функционал
* **Экран авторизации (Login Screen):** Ввод данных пользователя, маскировка пароля, обработка ошибок.
* **Экран регистрации (Register Screen):** Форма создания нового аккаунта с интерактивными полями ввода.
* **Валидация:** Проверка корректности введенных данных перед отправкой.
